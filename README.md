# 🌿 Keen Keeper

Keen Keeper is a beautifully designed web application that helps you stay connected with the people who matter most.
Track interactions, manage relationship goals, and never lose touch again.

---

## 🔗 Links

🌐 Live: https://sagorteasc-keen-keeper.vercel.app/ <br/>
📂 Repo: https://github.com/sagorteasc/PH-Assignment-07

---

## ✨ Features

- 🏠 Home page with banner and friend overview
- 👫 Friend list with status, tags, and last contact info
- 👤 Detailed friend profile with stats and actions
- ⚡ Quick check-in (Call, Text, Video) with toast notification
- 📜 Timeline to track all interactions
- 🔍 Filter timeline by interaction type
- 📊 Analytics page with Pie Chart (Recharts)
- 📱 Fully responsive design
- ❌ Custom 404 page

---

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS + DaisyUI
- **State Management:** React Context API
- **Charts:** Recharts
- **Icons:** React Icons + Lucide React
- **Notifications:** React Toastify

---

## 🎯 Purpose

This project was built as part of an assignment to demonstrate:

- Component-based architecture
- State management
- Dynamic UI rendering
- User interaction tracking
- Responsive design principles

## 🚀 Getting Started

Clone the project:

```bash
git clone https://github.com/sagorteasc/PH-Assignment-07.git
cd PH-Assignment-07
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
